# NovaX v1.0.0
基于angularJS的Web开发平台  
包含 完整的ng-whh指令库 和 Whh-Web开发平台
***
developer:  wangzheng
***
###ng-whh指令库

####whhGrid指令:  

    1.多种类型column,
    2.多列下拉,
    3.单列下拉,
    4.日期,
    5.日期时间,
    5.文件上传,
    6.command按钮,
    7.Excel导入导出, 
    8.分页
    9.crud tems记录
    10.query 
    11.勾选框 
    12.海量数据,edit change select等事件,本地数据源,远程数据源
    
####其他指令 

whhComboBox,WhhDropDown,WhhFileUpload,whhDateTimePicker,whhButton,whhDatePicker
